# README Standardization Checklist (SEO-01)

Use this checklist when updating each `terraform-do-modules` repository.

- [ ] Intro paragraph includes: Terraform, DigitalOcean, and concrete module capability.
- [ ] `source` and pinned `version` shown in first code snippet.
- [ ] terraform-docs tables present (`Requirements`, `Providers`, `Inputs`, `Outputs`).
- [ ] At least two examples (`basic` + `complete/production`).
- [ ] Architecture notes section added.
- [ ] Related modules section includes at least 3 valid cross-links.
- [ ] Broken badges/links removed.
- [ ] README passes markdown lint locally.
