# Keyword + Metadata Playbook (SEO-02)

## Primary Search Pattern

`terraform digitalocean <resource>`

## Title Pattern

`Terraform DigitalOcean <Resource> Module`

## README Intro Pattern

First paragraph should contain these terms naturally:
- Terraform
- DigitalOcean
- <resource>
- module
- infrastructure as code

## Repository Metadata

Set in each repo:
- Description: one sentence with Terraform + DigitalOcean + capability
- Topics (recommended):
  - terraform
  - terraform-module
  - digitalocean
  - infrastructure-as-code
  - <resource-specific-topic>

## Badge + Snippet Requirements

- Terraform Registry badge
- Release badge
- CI status badge
- Usage snippet with pinned module version

## Anti-Patterns

- Keyword stuffing in headings
- Generic descriptions like "Terraform module"
- Missing DigitalOcean mention
