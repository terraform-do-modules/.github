# terraform-do-modules

Open-source Terraform modules for DigitalOcean, maintained by [CloudDrove](https://clouddrove.com).

## Modules

| Module | Description |
|--------|-------------|
| [droplet](https://github.com/terraform-do-modules/terraform-digitalocean-droplet) | DigitalOcean Droplet management |
| [kubernetes](https://github.com/terraform-do-modules/terraform-digitalocean-kubernetes) | Kubernetes cluster management |
| [vpc](https://github.com/terraform-do-modules/terraform-digitalocean-vpc) | VPC networking |
| [database](https://github.com/terraform-do-modules/terraform-digitalocean-database) | Managed databases |
| [spaces](https://github.com/terraform-do-modules/terraform-digitalocean-spaces) | Object storage |
| [load-balancer](https://github.com/terraform-do-modules/terraform-digitalocean-load-balancer) | Load balancers |
| [firewall](https://github.com/terraform-do-modules/terraform-digitalocean-firewall) | Cloud firewalls |
| [domain](https://github.com/terraform-do-modules/terraform-digitalocean-domain) | DNS management |
| [monitoring](https://github.com/terraform-do-modules/terraform-digitalocean-monitoring) | Monitoring and alerts |
| [ssh-key](https://github.com/terraform-do-modules/terraform-digitalocean-ssh-key) | SSH key management |
| [labels](https://github.com/terraform-do-modules/terraform-digitalocean-labels) | Resource labeling |
| [app](https://github.com/terraform-do-modules/terraform-digitalocean-app) | App Platform |
| [cdn](https://github.com/terraform-do-modules/terraform-digitalocean-cdn) | CDN management |
| [certificate](https://github.com/terraform-do-modules/terraform-digitalocean-certificate) | SSL certificates |
| [container-registry](https://github.com/terraform-do-modules/terraform-digitalocean-container-registry) | Container registry |
| [nfs](https://github.com/terraform-do-modules/terraform-digitalocean-nfs) | NFS storage |

## Standards

All modules follow:
- Conventional commits for versioning
- Pre-commit hooks (antonbabenko/pre-commit-terraform)
- Terraform >= 1.10.0
- Apache 2.0 license
- Automated semantic releases
